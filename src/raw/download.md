## Download Subtitle

```bash

# cli
brew install yt-dlp

# subtitle only and remove timestamp
yt-dlp --skip-download --write-auto-subs --write-subs --sub-lang en --convert-subs \
https://www.youtube.com/watch?v=QJtPROVsePk --verbose
```
